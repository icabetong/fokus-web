# fokus-web

The source code of the website that showcases the Fokus application for Android.

## Built using

* ReactJS
* Gatsby
* Visual Studio Code

## Scripts

* ```gatsby develop``` - compiles and runs the development server
* ```gatsby build``` - compiles the source that will be used in production
